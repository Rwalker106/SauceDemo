# SauceDemo Test Suite

Automated end-to-end test suite for [SauceDemo](https://www.saucedemo.com/) built with **Playwright** and **pytest**.

## Project Structure

```
SauceDemo/
├── pages/
│   └── login_page.py       # Page Object Model for the login page
├── tests/
│   └── test_saucedemo.py   # Test cases
├── conftest.py             # pytest fixtures and browser setup
├── products.csv            # Exported product data (generated by tests)
├── products.json           # Exported product data (generated by tests)
├── report.html             # HTML test report (generated by pytest-html)
├── requirements.txt
└── .env                    # Credentials (not committed)
```

## Test Coverage

| Test                        | Description                                                                      |
| --------------------------- | -------------------------------------------------------------------------------- |
| `test_login_success`        | Verifies successful login redirects to the inventory page                        |
| `test_login_wrong_password` | Verifies correct error message for a bad password                                |
| `test_login_locked_user`    | Verifies locked-out users see the appropriate error                              |
| `test_login_errors`         | Parametrized test covering success, lockout, bad password, and empty credentials |
| `test_products_export`      | Scrapes all 6 products and exports them to `products.json` and `products.csv`    |
| `test_add_to_cart`          | Verifies adding the first item to the cart updates the cart badge                |

## Prerequisites

- Python 3.10+
- [Google Chrome](https://www.google.com/chrome/) (Playwright downloads its own Chromium by default)

## Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/Rwalker106/SauceDemo.git
   cd SauceDemo
   ```

2. **Create and activate a virtual environment**

   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # macOS/Linux
   source .venv/bin/activate
   ```

3. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   playwright install chromium
   ```

4. **Configure credentials**

   Create a `.env` file in the project root:

   ```env
   SAUCE_USER=standard_user
   SAUCE_PASSWORD=secret_sauce
   LOCKED_OUT_USER=locked_out_user
   PROBLEM_USER=problem_user
   PERFORMANCE_GLITCH_USER=performance_glitch_user
   VISUAL_USER=visual_user
   ERROR_USER=error_user
   ```

## Running the Tests

Run all tests:

```bash
pytest
```

Run with an HTML report:

```bash
pytest --html=report.html --self-contained-html
```

Run a specific test:

```bash
pytest tests/test_saucedemo.py::test_login_success
```

Run in headless mode (no browser window):

```bash
pytest --headed=false
```

## Tech Stack

- [Playwright for Python](https://playwright.dev/python/) — browser automation
- [pytest](https://docs.pytest.org/) — test framework
- [pytest-html](https://pytest-html.readthedocs.io/) — HTML test reports
- [python-dotenv](https://pypi.org/project/python-dotenv/) — environment variable management
